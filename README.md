# Brain_Tumor_ImageClassification01
This project implements a deep learning-based system to classify brain tumors from MRI images using the DenseNet121 architecture. The model categorizes images into four classes: Glioma, Meningioma, Pituitary tumor, and No Tumor.

This project focuses on detecting and classifying brain tumors from MRI images using a deep learning model based on DenseNet121. 
The system classifies images into four categories: Glioma, Meningioma, Pituitary tumor, and No Tumor.

The system includes preprocessing techniques like masking and normalization, statistical evaluation, and a web-based deployment for real-time prediction.

Objectives Develop a CNN model using DenseNet121 Perform accurate brain tumor classification 
Apply image masking to enhance tumor region detection Evaluate model using statistical metrics Deploy the model using a web application

Dataset Information Dataset link: https://data.mendeley.com/datasets/zwr4ntf94j/5 
Total Classes: 4 Glioma Meningioma Pituitary No Tumor

Model Details Base Model: DenseNet121 (Pretrained on ImageNet) Transfer Learning Applied Output Layer: Softmax (4 classes) Optimizer: Adam Loss Function: Categorical Crossentropy
